## help
- `cmd  --help` (explain command)

## keyboard shortcuts
- `CTRL + L` clear screen 

## directory
- `pwd` (print)
- `mkdir dir1` (create)
- `cd dir1` (change to)
- `cd..` (change up)
- `ls` (list files in cwd)

## file ops
- `touch file1` (create)
- `less file1` (view)
- `rm file1` (remove)
- `nano file1` (edit)
