## Methods
- keyword	`p {color: red}`
- hexadecimal	`p {color: ##ff0000}`  


- rgb	`p {color: rgb(255,0,0)}`
- rgba	`p {color: rgba(255,0,0,0.5)}`  


- [hsl](./hsl.md)	`p {color: hsl(0,100%, 50%)}`
- hsla	`p {color: hsla(240, 100%, 50%, 0.5)}`  


- opacity	`p {opacity: 0.5}`
