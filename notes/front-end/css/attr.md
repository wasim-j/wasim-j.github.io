# Select by Attribute

## [attr]
selects all elements which have `attr` set as an attribute

## [attr=val]
selects all elements which have `attr` set as an attribute  
AND if that **attribute's value is equal to `val`**

## [attr~=val]
selects all elements which have `attr` set as an attribute  
AND if that **attribute's space separated list of values is equal to `val`**
