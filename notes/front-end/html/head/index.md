# Head Element
Provides general information (metadata) about the document, including its title and links to its scripts and style sheets.

Elements that can be used inside a <head> element:
- &lt;title&gt;,
- [&lt;base&gt;](./base.md), 
- [&lt;link&gt;](./link.md), 
- &lt;style&gt;, 
- [&lt;meta&gt;](./meta.md), 
- &lt;script&gt;, 
- &lt;noscript&gt;, 
- &lt;template&gt;
